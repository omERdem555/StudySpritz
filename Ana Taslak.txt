STUDYSPRITZ ROADMAP (MASTER PLAN)

Projenin Amacı

StudySpritz; PDF, TXT, EPUB gibi metin tabanlı kitapları okuyabilen, okuma hızını artırmayı hedefleyen ve klasik okuma ile RSVP (Rapid Serial Visual Presentation) yöntemini aynı uygulama içinde sunan, ilerleme takibi yapan, istatistik üreten ve çok platformlu çalışan bir Flutter uygulamasıdır.

Desteklenen Platformlar

- Android
- iOS
- Windows
- Linux
- macOS
- Web

Tek kod tabanından tüm platformlar desteklenecektir.

==================================================
ANA HEDEFLER
==================================================

1. Dijital Kütüphane

Kullanıcı;

- kitap ekleyebilir
- kitap silebilir
- favorilere ekleyebilir
- kitapları listeleyebilir
- kitap geçmişini görebilir

--------------------------------------------------

2. Normal Okuma

Sayfa bazlı klasik okuma.

Özellikler;

- devam et
- kaldığı sayfaya git
- kaldığı kelimeye git
- sayfa ilerlet
- sayfa geri al
- otomatik kayıt

--------------------------------------------------

3. RSVP Okuma

Tek kelime gösterimli hızlı okuma.

Özellikler;

- WPM ayarı
- pause
- resume
- ileri
- geri
- anchor kaydı
- normal okumaya geri dönebilme

--------------------------------------------------

4. Bookmark Sistemi

Kullanıcı;

- bookmark ekleyebilir
- not ekleyebilir
- bookmark silebilir
- bookmark listesi görebilir

--------------------------------------------------

5. İstatistik Sistemi

Takip edilecek veriler;

- toplam okuma süresi
- toplam okunan kitap
- toplam kelime
- toplam sayfa
- günlük kullanım
- peak WPM
- average WPM
- oturum sayısı

--------------------------------------------------

6. Ayarlar

Kullanıcı;

- tema
- font
- font büyüklüğü
- satır aralığı
- animasyon
- varsayılan WPM
- varsayılan okuma modu

ayarlarını değiştirebilir.

--------------------------------------------------

7. Çok Platform Desteği

Platformlara özel dosya erişimleri.

- Web
